# Yijia Wang Portfolio

A bilingual personal portfolio for Yijia Wang / 王奕嘉, built as a simple Jekyll static site for GitHub Pages.

## Local Preview with Jekyll

Install Ruby first if `ruby -v` does not work in your terminal.

```bash
bundle install
bundle exec jekyll serve
```

Then open the local address shown in the terminal, usually:

```text
http://127.0.0.1:4000/
```

## Editing Content

- English pages are in `en/`
- Chinese pages are in `zh/`
- Shared layout files are in `_layouts/` and `_includes/`
- Main styles are in `assets/css/portfolio.css`
- Resume PDF is at `assets/pdf/Yijia_Wang_Resume.pdf`

## Deployment

This structure is ready for GitHub Pages. Push the repository to GitHub, then enable Pages from the repository settings.
